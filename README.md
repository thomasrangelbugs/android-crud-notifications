# Android CRUD + Notifications

> App Android academico com SQLite, menu de opcoes e notificacoes automaticas.

## Sobre o projeto

Projeto academico Android em Java com cadastro e listagem de conteudos em SQLite. Menu de opcoes no canto superior direito navega entre telas de Listagem e Cadastro. Dispara notificacao do sistema sempre que o total de itens salvos atinge multiplo de 10, com solicitacao de permissao no Android 13+.

## Funcionalidades principais

- Menu de opcoes com navegacao Listagem/Cadastro
- Persistencia local em SQLite
- CRUD completo de conteudos
- Notificacao automatica a cada 10 registros
- Permissao de notificacao Android 13+
- Notification Channels e Material Design

## Tecnologias utilizadas

- Android Java
- SQLite
- Notification Channels
- Material Design
- Gradle

## Como executar

Abra a pasta no Android Studio, sincronize Gradle e execute no emulador.

## Deploy / Demonstracao

App Android - gere APK via Android Studio.
